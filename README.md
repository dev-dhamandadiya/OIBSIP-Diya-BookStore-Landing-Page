# 📚 BookStore Landing Page

A modern and responsive BookStore Landing Page built using HTML and CSS. This project is designed to provide book lovers with an engaging platform to explore different book categories, discover new reads, and connect with the world of knowledge.

## 🚀 Project Overview

The BookStore Landing Page features a clean and attractive user interface that highlights various book collections and services. The website is designed with simplicity, readability, and user experience in mind.

## ✨ Features

- Responsive Navigation Bar
- Attractive Hero Section
- About BookStore Section
- Book Categories Showcase
- Why Choose Us Section
- Contact Us Form
- Professional Footer
- Smooth Hover Effects
- Clean UI Design

## 🛠️ Technologies Used

- HTML5
- CSS3
